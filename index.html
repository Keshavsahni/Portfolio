Simon Says

created a game using JavaScript. This game is designed to aid individuals with low memory power. It features four colored lights and includes ten levels. The game works by flashing a sequence of colored lights, which users then have to repeat in the correct order. As the levels progress, users are challenged to recall longer sequences, including the current and previous lights. While traditionally a physical game, I tried adapting it for online use to enhance accessibility
